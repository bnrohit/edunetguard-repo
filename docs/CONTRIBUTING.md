# Contributing to EduNetGuard

Thank you for your interest in contributing! We welcome contributions from schools, developers, and IT professionals.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/edunetguard.git`
3. Create a branch: `git checkout -b feature/your-feature-name`

## Development Setup

```bash
# Start in development mode
docker compose -f docker-compose.yml -f docker-compose.dev.yml up

# Or run frontend/backend separately
cd frontend && npm start
cd backend && npm run dev
```

## Guidelines

- Follow existing code style
- Write clear commit messages
- Test your changes locally
- Update documentation if needed
- Be respectful and inclusive

## Reporting Issues

Please include:
- EduNetGuard version
- Docker version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Code of Conduct

Be kind, be helpful, be patient. We're all here to make school IT better.
